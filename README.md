# 📁 Auto-Organizador de Arquivos

Automação inteligente para organização de arquivos usando **Python** e **Watchdog**.

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🎯 Funcionalidades

- ✅ Organização automática por tipo de arquivo
- ✅ Monitoramento em tempo real (modo watch)
- ✅ Regras customizáveis via YAML
- ✅ Detecção de arquivos grandes (+100MB)
- ✅ Sistema completo de logs
- ✅ Evita conflitos de nome de arquivos
- ✅ Execução única ou contínua
